# Modelo de Etapas — la evolución de una marca personal

> El corazón estratégico de FARO. Una marca personal no es un estado, es un **tránsito**, y cada etapa tiene su propio juego: objetivos, métricas, errores típicos y un criterio claro de graduación. El error más común del oficio es aplicar tácticas de una etapa estando en otra (ej: montar un funnel de venta sin tener confianza construida, o seguir "rompiendo el hielo" cuando ya hay autoridad que monetizar).
>
> Base: síntesis del oficio construida sobre evidencia documentada (modelo Lenny Rachitsky: consistencia → contenido original → audiencia propia → monetización tardía; hallazgo clave de monetización para expertos: bastan 500-2.000 contactos cálidos para vender servicios — ver `playbooks/monetization.md`). El modelo de 5 etapas como tal es **[criterio del oficio]**: una herramienta de navegación, no una ley natural.

---

## Las 5 etapas

### E0 · Chispa — "Sé que quiero, pero no he empezado"

**Señales:** nada publicado con intención de marca (o publicaciones aisladas sin dirección). Miedo a la cámara o a la opinión ajena. Ideas dispersas, sin nicho declarado.

| | |
|---|---|
| **Objetivo único** | Existir: definir identidad mínima y publicar las primeras piezas imperfectas |
| **Foco** | Claridad: quién soy, a quién le hablo, qué defiendo (manifiesto). Romper el hielo frente a la cámara/el teclado |
| **Cadencia** | La que rompa el hielo: 1 pieza/semana ya es victoria |
| **Métrica norte** | Piezas publicadas (sí, cantidad — la calidad viene después) |
| **Errores típicos** | Perfeccionismo paralizante · comprar equipamiento antes de publicar · abrir 4 canales a la vez · esperar "tener la estrategia perfecta" |
| **El sistema frena** | Multicanal, monetización, branding visual sofisticado |
| **🎓 Graduación a E1** | Identidad mínima definida (perfil + voz + 3-5 pilares) **y** 5-10 piezas publicadas **y** el hielo roto (publicar ya no paraliza) |

### E1 · Cimientos — "Publico, pero casi nadie me ve"

**Señales:** publica con cierta regularidad. Audiencia pequeña (decenas-cientos). Todavía buscando su voz; engagement bajo y errático.

| | |
|---|---|
| **Objetivo único** | Consistencia que entrene al algoritmo y a la audiencia: que te reconozcan un tema y una voz |
| **Foco** | 1 canal **ancla** (donde está tu audiencia objetivo) + 1 de apoyo, no más. Disciplina de nicho: los algoritmos premian la consistencia temática. Encontrar la voz publicando, no teorizando |
| **Cadencia** | Sostenible y declarada: ej. 2-3 piezas/semana en el ancla. La regla: la cadencia que puedes mantener 6 meses |
| **Métrica norte** | Racha de consistencia (semanas cumpliendo el plan) + señales de resonancia (comentarios, guardados, mensajes privados) |
| **Errores típicos** | Cambiar de nicho por ansiedad · perseguir el formato viral del momento · comparar su capítulo 1 con el capítulo 20 de otros · abrir TikTok porque "hay que estar" |
| **El sistema frena** | Más canales, cambios de rumbo impulsivos, obsesión con seguidores |
| **🎓 Graduación a E2** | 3+ meses de cadencia cumplida **y** patrón identificable de qué piezas resuenan (hay datos para duplicar) **y** primeras personas que te reconocen el tema |

### E2 · Tracción — "Algo está funcionando"

**Señales:** engagement crece con patrón reconocible. Llegan seguidores del nicho correcto. Algunas piezas despegan y se sabe por qué.

| | |
|---|---|
| **Objetivo único** | Duplicar lo que funciona y convertir alcance prestado en **audiencia propia** |
| **Foco** | Análisis de resonancia (qué pilares/formatos/hooks funcionan → más de eso). Encender el motor de reutilización (1 pieza ancla → N derivados; es lo que hace sostenible el multicanal). Iniciar lista de email/newsletter: el activo que no depende de algoritmos |
| **Cadencia** | La de E1 + derivados sistemáticos. Abrir 2º-3º canal solo vía reutilización, nunca producción nueva por canal |
| **Métrica norte** | Crecimiento de lista propia + tasa de resonancia (guardados/compartidos por pieza) |
| **Errores típicos** | Dispersión temática ahora que "hay público" · vanity metrics (likes sobre conversaciones) · monetizar antes de tiempo y quemar confianza · no documentar qué funcionó (sin datos no se duplica) |
| **El sistema frena** | Ofertas de venta prematuras, producción nueva por canal (en vez de derivar) |
| **🎓 Graduación a E3** | Lista propia creciendo **y** te citan/preguntan como referencia en el tema **y** llegan oportunidades entrantes (invitaciones, consultas, colaboraciones) |

### E3 · Autoridad — "Me empiezan a buscar"

**Señales:** referente emergente del nicho. Invitaciones a podcasts/eventos/colaboraciones. Mensajes pidiendo ayuda o servicios. La audiencia confía.

| | |
|---|---|
| **Objetivo único** | Convertir autoridad en relación profunda y en las **primeras ofertas** |
| **Foco** | Pieza ancla de profundidad (newsletter sólido, podcast, video largo, serie original con data propia — el "proyecto épico"). Lead magnet que filtre y capture al héroe correcto. Red estratégica: entrevistas/colaboraciones elegidas por valor de relación, no por fama. Primeras ofertas de baja barrera (workshop, plantilla, mentoría puntual) |
| **Cadencia** | Menos volumen, más profundidad. El ancla semanal/quincenal manda; los derivados llenan el resto |
| **Métrica norte** | Conversaciones de valor iniciadas (DMs, respuestas de email, llamadas) + primeras ventas |
| **Errores típicos** | Miedo a vender ("aún no estoy listo") — recordar: para servicios bastan 500-2.000 contactos cálidos, no fama · regalar todo para siempre · aceptar toda invitación sin criterio · descuidar la consistencia que lo trajo hasta aquí |
| **El sistema frena** | Falsa modestia que postergue la oferta, colaboraciones sin encaje estratégico |
| **🎓 Graduación a E4** | Ofertas vendidas con regularidad (no por suerte) **y** demanda entrante estable **y** claridad de qué escalón de la escalera de valor funciona |

### E4 · Motor — "Vivo (en parte) de esto"

**Señales:** monetización recurrente. La marca genera demanda sola. El cuello de botella ya no es la audiencia: es el tiempo.

| | |
|---|---|
| **Objetivo único** | Sistematizar: que la marca funcione como motor, no como sprint permanente |
| **Foco** | Escalera de valor completa (gratis → bajo → medio → alto). Comunidad como foso defensivo. Delegación/automatización de producción (edición, programación, derivados). Proteger el contenido de autoridad que lo originó todo |
| **Cadencia** | La que el sistema (no el héroe agotado) pueda sostener: batching, equipo, herramientas |
| **Métrica norte** | Ingresos por la marca + horas propias invertidas (la división importa) |
| **Errores típicos** | Hacer todo a mano por costumbre · dejar de crear el contenido profundo "porque ya vendo" (erosiona la autoridad que sostiene la venta) · escalar ofertas sin proteger la calidad |
| **El sistema frena** | Operación artesanal en tareas delegables, abandono del contenido raíz |
| **🎓 Horizonte** | La marca como activo: comunidad viva, ingresos diversificados, opcionalidad (lanzar producto, libro, fondo, lo que venga) |

## Los modificadores transversales

La etapa define el *juego*; estos ejes definen *cómo se juega*. Viven en `brand/profile.md` y modifican toda recomendación:

| Eje | Valores típicos | Qué cambia |
|-----|-----------------|------------|
| **Objetivo de negocio** | Vender servicios (consultoría/mentoría) · Empleabilidad/carrera · Lanzar producto · Construir comunidad/influencia | El CTA de fondo, la oferta, qué métrica importa. Quien vende servicios necesita pocos clientes correctos; quien busca empleabilidad necesita visibilidad ante decisores; quien lanza producto necesita lista |
| **Tipo de profesional** | Consultor/independiente · Empleado/ejecutivo · Fundador · Académico/especialista · Creador | El riesgo reputacional, la frontera empresa-persona, los temas hablables, la urgencia de monetizar |
| **Audiencia** | B2B · B2C · Pares de industria · Mixta | El canal ancla (B2B → LinkedIn; B2C amplio → IG/TikTok/YouTube), el registro, los formatos |
| **Tiempo real disponible** | 2-3 h/sem · 5-8 h/sem · 10+ h/sem | La cadencia, cuánto multicanal es honesto, cuánto delegar |
| **Comodidad frente a cámara** | Cero · En proceso · Natural | El mix texto/video del arranque y la rampa de exposición |
| **Punto de partida** | De cero · Reactivando · En transición de nicho | Si se construye, se retoma o se re-posiciona (la transición de nicho requiere puente narrativo, no borrón y cuenta nueva) |

## Cómo usa FARO este modelo

1. **Diagnóstico** (`skills/assess-stage.md`): preguntas sobre señales observables → etapa propuesta → acuerdo con el usuario → se escribe en `brand/state.md`.
2. **Comportamiento**: la matriz "empuja/frena" del `CLAUDE.md` se activa según la etapa.
3. **Graduación**: en cada revisión mensual se chequean los criterios de graduación. Subir de etapa es un evento: se celebra, se registra en `journal.md` y se recalibra la estrategia.
4. **Regresión honesta**: si la vida golpea y la cadencia muere 2+ meses, se puede bajar el foco a la etapa anterior sin drama. El modelo sirve a la persona, no al revés.
