# System Learnings — aprendizajes sobre FARO (no sobre tu marca)

```yaml
faro_version_origin: 1.1.0   # versión con la que se instaló esta copia (no cambiar a mano)
faro_version_current: 1.1.0  # versión actual de esta copia (se actualiza al actualizar el sistema)
modified: false              # true si se modificó core/, agents/, skills/ o CLAUDE.md
last_sent: null              # fecha del último envío de feedback (lo gestiona FARO)
```

> Lo escribe FARO durante el uso (skill `system-feedback`), **siempre anonimizado y sin datos de la marca**. Tú puedes escribir aquí directamente también. Revisa este archivo antes de enviarlo — es tuyo.
